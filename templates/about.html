About EngageTrack

EngageTrack is a real-time student engagement detection system powered by YOLOv11 deep learning technology. It helps educators monitor and analyze student engagement during classes, providing valuable insights to improve teaching effectiveness.

Key Features

Real-Time Detection
Live video analysis with instant feedback on student engagement levels
Student Tracking
Consistent ID assignment for individual student analysis throughout sessions
Smart Alerts
Automatic alerts when sleepy or disengaged students are detected
Detailed Analytics
Comprehensive reports and historical data for engagement analysis
Database Storage
PostgreSQL database for reliable data storage and retrieval
High Performance
Optimized for fast FPS while maintaining detection accuracy

Technology Stack

  • YOLOv11: State-of-the-art object detection for real-time engagement analysis
  • Flask: Lightweight Python web framework for the backend
  • PostgreSQL: Robust relational database for data persistence
  • OpenCV: Computer vision library for video processing
  • SQLAlchemy: Python SQL toolkit and ORM

How It Works

  1. The system captures live video feed from your camera
  2. YOLOv11 model analyzes each frame to detect students and their engagement states
  3. Students are assigned consistent IDs for tracking throughout the session
  4. Engagement data is classified as: Engaged, Neutral, Bored, or Sleepy
  5. Real-time statistics are displayed and stored in the database
  6. Alerts are triggered when concerning patterns are detected
  7. Session data is saved for later analysis and reporting

Privacy & Ethics

EngageTrack is designed with privacy in mind. All video processing happens locally, and only engagement statistics are stored in the database. No video recordings or facial recognition data are saved. The system should be used transparently with student consent and in accordance with institutional policies.
